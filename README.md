# LiveStock Track Monorepo

- `frontend/` — Vite + React + Tailwind dashboard UI
- `backend/` — Node.js + Express + MongoDB auth API

## Frontend
```bash
cd frontend
npm install   # first time only
npm run dev
```

## Backend
```bash
cd backend
npm install   # already installed
npm run dev
```

Set environment variables for the backend using `.env` in `backend/` (copy from `.env.example`).
